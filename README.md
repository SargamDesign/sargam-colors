# Sargam Colors
An open-source color system for crafting delightful, accessible websites and apps.

```
.
├── tokens/
│   ├── global/
│   │   └── global.json
│   └── themes/
│       ├── dark/
│       │   └── dark.json
│       └── light/
│           └── light.json
└── dist/
    └── themes/
        ├── light.css
        └── dark.css
```

### Themes

#### Light
- Build theme: `npm run build:tokens -- --theme=light`
- [Get a Figma copy](https://www.figma.com/community/file/1161992682973418812)
